# 2026-04-01

### 3. Konfigurér Periodic Notes 
Nu skal vi fortælle Obsidian, at den skal bruge denne skabelon: 
1. Gå til **Settings > Periodic Notes > Daily Notes**. 
2. **Template file:** Vælg `docs/templates/Journal-Skabelon.md`. 
3. **Note folder:** Vælg `docs/journal`. 
4. **Folder format:** Hvis du vil have orden, kan du skrive `YYYY/MM` (så laver den automatisk mapper for år og måned).
